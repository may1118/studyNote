# Canvas和SVG区别

| canvas                       | SVG                                                       |
| ---------------------------- | --------------------------------------------------------- |
| 一整幅画布                   | 绘制出来的都是独立的`DOM`节点，能够方便绑定事件或用来修改 |
| 一张图片，放大会失真或者锯齿 | 适量图形，后期可以修改参数来自由放大缩小、不会是真和锯齿  |
