# link和@import区别

| link                                                     | @import                   |
| -------------------------------------------------------- | ------------------------- |
| 属于HTML标签，除了加载css之外还能定义rel属性完成其他功能 | CSS提供，只能加载CSS      |
| 页面加载的时候会被同时加载                               | 等CSS页面加载完之后再加载 |
| 没有兼容问题                                             | IE５以上才被识别          |
