# 行内元素 块级元素 空元素

## 行内元素

- `span` 
- `a` 
- `img`
- `select`
- `input` 

```html
    <select name="" id="">
      <option value="a">a</option>
      <option value="b">b</option>
      <option value="c">c</option>
    </select>
```

## 块级元素

- `div`
- `p`
- `h1-h5`
- `ul` `ol` `li`

## 空元素

- `img`
- `br`
- `input`
- `link`
- `hr`

## 块级元素和行类元素区别

- 块级元素独占一行，可以设置宽高
- 行类元素不独占一行，不可以设置宽高
- 行类元素作用的`padding` 和 `margin` `border`有有效
