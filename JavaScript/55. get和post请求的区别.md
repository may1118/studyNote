# get和post请求的区别

| GET                                      | POST                   |
| ---------------------------------------- | ---------------------- |
| 可缓存                                   | 无缓存                 |
| 无害                                     | 刷新数据重新提交       |
| 能保存到浏览器历史中                     | 不会保存到浏览器历史中 |
| 安全性差                                 | 更安全                 |
| 数据在URL中，对所有人可见（URL进行传参） | 数据不会显示在URL中    |
| 浏览器显示了URL长度，所有数据传输优先    | 数据传输没有限制       |
