# attribute和property

| attribute                                 | property                              |
| ----------------------------------------- | ------------------------------------- |
| `DOM`元素在文档中作为`HTML`标签拥有的属性 | `DOM`元素在`JS`中作为对象拥有的属性   |
| 不可变                                    | 可变                                  |
| 可以自定义                                | 不可以自定义                          |
| 保持`HTML`的初始值                        | 可能变化（input的输入，保持用户输入） |
