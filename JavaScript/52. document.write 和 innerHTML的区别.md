# document.write 和 innerHTML的区别

| document.write                                    | innerHTML                                             |
| ------------------------------------------------- | ----------------------------------------------------- |
| 直接写入到页面的内容流，使用之前需要document.open | DOM元素的一个属性，可以精确到某一个具体的元素进行更改 |
| 重写整个document，导致重绘整个页面                | 重写某个元素，只会重绘页面一部分                      |
